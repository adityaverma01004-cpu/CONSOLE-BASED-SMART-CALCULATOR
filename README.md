Student Utility Toolkit

A Python-based mini project that includes multiple utility programs for basic calculations, student performance analysis, and expense tracking.

Features
1. Calculator Program
Supports arithmetic operations:
Addition (+)
Subtraction (-)
Multiplication (*)
Division (/)
Modulus (%)
Power (^)
Accepts multiple user inputs.
Performs calculations based on selected operations.
2. Student Grade Analyzer
Accepts marks for multiple students.
Assigns grades automatically:
EX (90–100)
A (80–89)
B (70–79)
C (60–69)
FAIL (<60)
Displays:
Minimum score
Maximum score
Average score
Total score
3. Expense Tracker
Records daily expenses.
Alerts when expenses exceed a predefined limit.
Calculates:
Highest expense
Day with maximum expense
Average expense
Technologies Used
Python 3
NumPy
